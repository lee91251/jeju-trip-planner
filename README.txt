제주 여행 플래너 (Jeju Trip Planner)
=====================================

🌐 인터넷 주소 (폰·PC 어디서나 접속)
   https://lee91251.github.io/jeju-trip-planner/

📁 파일 설명 (지금 쓰는 버전)
   index.html ......... 웹사이트 메인 화면 (디자인)
   map.js ............. 지도·검색·일정·주변발견 기능
   data.js ............ 제주 추천 장소 49곳 + 섬 모양 데이터
   places_jeju.json ... 관광공사 제주 관광지 1,038곳 + 사진
   preset_img.js ...... 추천 장소들의 정확한 사진
   .nojekyll .......... 배포 설정 파일 (건드리지 마세요)

   _archive_v1/ ....... 옛 일러스트 버전 백업 (지금은 안 씀)

✏️ 수정하려면
   Claude Code에게 말하면 자동으로 바뀌고,
   1~2분 뒤 위 인터넷 주소에 반영됩니다.
